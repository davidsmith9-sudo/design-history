# 大衛·阿賈耶

## 簡述

大衛·法蘭克·阿賈耶爵士，OM，OBE，RA，為一名加納裔英國籍建築師。他的作品遍及世界各地，其中較有名的包括位於美國華盛頓哥倫比亞特區的非裔美國人歷史和文化國家博物館，以及位於俄羅斯莫斯科的斯科爾科沃管理學院。


## 生平
1966 生於坦尚尼亞達累斯薩拉姆,父親是加納外交官 — 童年隨父親派駐多國(肯亞、埃及、葉門、黎巴嫩)。9 歲移居英國倫敦。1990 倫敦藝術大學 South Bank University 建築學士,1993 皇家藝術學院(RCA)碩士。1994-2000 與 William Russell 合夥,2000 設立 Adjaye Associates。2000 代前期設計多棟英國藝術家朋友的住宅(Elektra House、Lost House)。2007-09 個人破產危機(過度擴張 + 經濟危機),透過 Sir Norman Foster 接濟與企業重組度過。2009 中標華盛頓 DC 史密森尼國立非裔美國人歷史與文化博物館(NMAAHC),2016 完工開幕,成為其代表作。2017 獲英女王封爵。2023 因公司前員工性騷擾指控陷入爭議,部分項目暫停,公司重組中。

## 設計理念
- **「非洲現代主義」(African Modernism)** — 從非洲傳統建築(尤其西非如約魯巴皇宮)發展現代語言,反對「現代主義 = 歐洲」教條
- **物質與光的對話** — 大量使用青銅、木材、磚等溫暖材料,反對國際樣式的純白冷感
- **後殖民立場** — 公開談論建築師作為「黑人 + 加納-英國 + 多國成長」身分如何影響設計
- **公共建築為核心** — 多數作品為博物館、社區中心、住宅,而非企業總部
- **與[[解構主義]]距離** — 不追求蓋瑞式自由曲面,更接近合理但有色彩感的長方體

## 代表作品
- **Idea Stores 系列**(2004-, 倫敦東區)— 圖書館 + 社區中心 hybrid
- **諾貝爾和平中心**(2005, 奧斯陸)— 國際曝光起點
- **Rivington Place**(2007, 倫敦)— 非洲與亞洲藝術中心
- **Sugar Hill 住宅**(2014, 紐約哈林區)— 124 戶廉租住宅 + 兒童博物館
- **史密森尼國立非裔美國人歷史與文化博物館**(NMAAHC, 2016, 華盛頓 DC)— 三層青銅外殼如非洲皇冠,設計圖騰
- **Aïshti Foundation**(2015, 貝魯特)
- **諾華園區**(Novartis, 巴塞爾)
- **未來計畫**:倫敦 Holocaust Memorial(規劃中)、迦納大教堂(2020-, 規劃中)

## 影響
- **黑人建築師可見度** — 21 世紀第一位獲爵位 + 全球委託級的非洲裔建築師
- **2017 獲爵士頭銜** — 加納-英國雙重身分,英國第一位非洲裔被封爵的建築師
- **2021 普利茲克獎** — 提名熱門但未獲(Anne Lacaton + Jean-Philippe Vassal 獲獎)
- **NMAAHC 的文化意義** — 開幕後成為華盛頓 DC 最受參觀博物館之一,被視為「美國非裔身份的視覺宣言」
- **2023 醜聞** — 性騷擾指控讓他失去多個項目(包括聯合國總部翻修諮詢),公司至今重組中
- **教育倡議** — 創辦 David Adjaye Foundation 支持非洲建築教育
- **批評** — 部分人質疑其「後殖民立場」與「為西方精英客戶服務」的張力
