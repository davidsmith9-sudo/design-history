# 設計史總時間軸

完整年代序，自動依 `year_start` 排序。

## 所有流派（年代序）

```dataview
TABLE
  era AS "時代",
  region AS "地區",
  year_start AS "起",
  year_end AS "迄",
  field AS "領域"
FROM "40-流派"
WHERE type = "流派"
SORT year_start ASC
```
