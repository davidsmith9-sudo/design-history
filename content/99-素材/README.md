# 99-素材

這個資料夾放所有**外部檔案**：圖片、PDF、影片等。

## 建議子資料夾結構

```
99-素材/
├── images/      ← 一般圖片
│   ├── 流派/
│   ├── 人物/
│   └── 作品/
├── pdfs/        ← 文獻 PDF
└── refs/        ← 其他參考資料
```

## 在筆記中引用圖片

```markdown
![[99-素材/images/作品/包浩斯校舍.jpg]]
```

或用相對路徑：

```markdown
![[包浩斯校舍.jpg]]
```

只要檔名在整個 vault 是唯一的，Obsidian 會自動找到。

## 版權注意

爬到的圖片若要公開（例如階段 6 發布網站），請確認：
- 維基共享資源（Wikimedia Commons）：多為公有領域或 CC 授權，可用
- 其他來源：須個別確認授權，建議只在本地筆記使用
