# UI/UX 設計史

> **核心子領域**　GUI、互動設計、使用者體驗、服務設計

人機介面與使用者體驗設計的歷史。從早期命令列到 GUI、行動觸控、語音、AR/VR、AI。

## 此領域的流派

UI/UX 屬於[[當代設計]]領域，傳統「流派」概念較少；多以**設計系統 / 公司風格**取代：

- Macintosh / 早期 Apple HIG（1984–）
- Microsoft Windows 風格（90s–）
- Apple Human Interface Guidelines（2007 iPhone 後）
- Google Material Design（2014–）
- 扁平化設計（2010s）
- Neumorphism / Glassmorphism（2020s）

## 設計特徵

- 使用者中心（user-centered）— 從技術導向轉為人本
- 一致性（consistency）— 跨頁面、跨平台
- 回饋（feedback）— 點擊、載入、錯誤狀態
- 可達性（accessibility）— 色盲、螢幕閱讀器、鍵盤導航
- 設計系統（design system）— 元件可重用、設計與工程協作
- 互動模式（pattern）— 慣例優於創新

## 歷史演變

- **1960s** — Sketchpad（Sutherland）、滑鼠原型（Engelbart）、NLS demo
- **1970s** — Xerox PARC：Alto、Smalltalk、GUI 雛形
- **1984** — Macintosh 普及 GUI、桌面隱喻
- **1990s** — Web 設計誕生、瀏覽器戰爭、HTML+CSS
- **2007** — iPhone 與多點觸控介面
- **2010s** — 扁平化、響應式設計、Material Design
- **2020s** — AI 介面、語音、3D／空間運算（Vision Pro）
