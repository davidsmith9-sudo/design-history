# 字體設計史

## 定義
活字、書法、現代字型設計的歷史 — 從手寫到金屬活字到當代數位字型。字體是設計史中與「閱讀」最直接連結的領域,影響範圍從每一本書、每一張海報到每一個螢幕介面。一個字體可以承載一個時代的審美、技術、政治、商業立場。

## 歷史脈絡

### 古代 — 書寫系統(前 3000-公元 500)
- **兩河楔形** — 蘇美刻寫於泥板,前 3200
- **埃及象形文字** — 神聖意義 + 圖像化,前 3000
- **腓尼基字母** — 前 1050,影響後續希臘、拉丁、希伯來、阿拉伯字母
- **古希臘字母** — 前 800,西方字母奠基
- **羅馬大寫(Capitalis Monumentalis)** — 圖拉真柱(113)上的銘文,當代襯線字體的祖先
- **漢字** — 商代甲骨文(前 1600)→ 篆書 → 隸書(漢)→ 楷書(隋唐)→ 宋體(明清雕版印刷)

### 中世紀手抄本(500-1450)
- **Uncial 字體** — 圓潤希臘風,聖經抄本
- **加洛林小寫(Carolingian minuscule)** — 800 由 Charlemagne 推動,後成歐洲標準
- **哥德式黑體(Blackletter / Gothic)** — 12 世紀後北歐主流,古騰堡聖經用此

### 古騰堡到 18 世紀(1450-1800)
- **1450 古騰堡聖經** — 西方第一本活字印刷書,字體模仿哥德手抄
- **1495 Aldus Manutius + Francesco Griffo** — 威尼斯,設計 Bembo 字體 + 第一個斜體(italic)
- **Garamond**(Claude Garamond, 1530)— 法國,文藝復興襯線字體經典,至今仍廣用
- **Caslon**(William Caslon, 1722)— 英國,英語世界 18 世紀標準,美國獨立宣言印刷字體
- **Baskerville**(John Baskerville, 1757)— 過渡襯線
- **Didone(Modern)字體** — Bodoni(1788)、Didot(1784),極端粗細對比,新古典時代

### 19 世紀 — 海報字體爆炸
- **平版印刷** + 木刻活字 → 大字海報需求
- **第一個無襯線(Sans-serif)** — William Caslon IV(1816),最早叫「Egyptian」或「Grotesque」
- **木刻字體** — 美國西部風格,各種粗黑、陰影、3D 字體
- **Akzidenz-Grotesk**(1898, Berthold)— 德國,後成為 [[Helvetica]] 的前身

### 20 世紀前期 — 現代字體革命
- **Futura**(Paul Renner, 1927)— [[包浩斯]]精神,純幾何構成
- **Gill Sans**(Eric Gill, 1928)— 英國人文無襯線
- **Times New Roman**(Stanley Morison, 1932)— *Times* 報紙專屬,後成全球辦公標準
- **[[揚·奇肖爾德]]《新印刷術》**(1928)— 主張無襯線 + 不對稱 + 系統化

### 戰後 — 瑞士國際主義(1950-70)
- **[[Helvetica]]**([[馬克斯·米丁格]] + Hoffmann, 1957)— Neue Haas Grotesk
- **[[Univers]]**(Adrian Frutiger, 1957)— 21 字重數字編號系統
- **Frutiger**(Adrian Frutiger, 1976)— 為巴黎戴高樂機場設計,後成機場標識字體標準
- **Sabon**(揚·奇肖爾德, 1967)— 古典襯線,書籍排版主流

### 數位字體革命(1980-)
- **PostScript Type 1**(1984, Adobe)— 第一個可縮放數位字體技術
- **TrueType**(1991, Apple/Microsoft)— 取代 PostScript 為主流
- **Adobe Originals**(1989-)— Adobe Garamond、Minion、Myriad
- **Hoefler & Frere-Jones**(1989-)— Mercury、Gotham、Archer
- **Verdana + Georgia**(Matthew Carter, 1996)— Microsoft 委託,為螢幕優化
- **Open Source 字體** — Google Fonts(2010-)、Adobe Fonts、SIL Open Font License
- **可變字型(Variable Font)**(2016-)— 一個檔案多種字重 / 字寬 / 斜度

### 中文字體
- **宋體** — 明清雕版印刷期間發展出橫細豎粗、有襯線的「印刷體」,日文稱「明朝體」
- **黑體** — 民國時期受日本影響發展,1960 後成中文無襯線標準
- **數位中文字型** — 文鼎 → 文泉驛(2007 開源)→ Source Han Sans(思源黑體, 2014 Adobe + Google)、Noto Sans CJK
- **當代** — Justfont、文鼎、Arphic、字嗨等繁體中文字型公司活躍
