# UI/UX 設計史

## 定義
人機介面與使用者體驗設計的歷史 — 從早期命令列到 GUI、行動觸控、語音、AR/VR、AI 介面。是設計史中最年輕(60 年)卻發展最快的領域,1980 代後成為設計學科主要新興方向。

## 歷史脈絡

### 雛形期(1945-1970)— 想像力先於技術
- **1945 Vannevar Bush《As We May Think》** — *The Atlantic Monthly* 雜誌文章,提出 Memex 構想:可超連結的書桌型機器,是「網頁」的雛形
- **1963 Ivan Sutherland Sketchpad** — MIT 博士論文,第一個圖形使用者介面(GUI)+ 物件導向繪圖
- **1968 道格拉斯·恩格爾巴特「Mother of All Demos」** — 史丹佛研究所現場演示滑鼠、視窗、超連結、視訊會議、共筆文件 — 之後 50 年所有 UI 的雛形
- **1972 Smalltalk @ Xerox PARC** — Alan Kay 主導,物件導向程式 + 圖形 GUI 整合

### Xerox PARC 黃金期(1970-1980)
- **1973 Xerox Alto** — 第一台個人電腦 GUI 系統,但未商品化
- **1979 Steve Jobs 訪問 PARC** — 看到 Alto 後決定把 GUI 帶到蘋果
- **1981 Xerox Star** — 商品化 GUI 工作站,但售價 7 萬美元失敗
- **WYSIWYG(What You See Is What You Get)** — PARC 確立的關鍵概念
- **桌面比喻(Desktop metaphor)** — 檔案夾、垃圾桶、文件等,把實體辦公比喻數位化

### GUI 大眾化(1984-1995)
- **1984 Apple [[Macintosh]]** — Susan Kare 圖示、Bill Atkinson GUI、HyperCard
- **1985 Microsoft Windows 1.0** — 起初為 MS-DOS 的圖形 shell
- **1989 Tim Berners-Lee 在 CERN 發明 World Wide Web**
- **1990 NeXT 上開發第一個瀏覽器** — 由 Berners-Lee 完成
- **1993 NCSA Mosaic** — 第一個圖形網頁瀏覽器
- **1995 Windows 95** — 個人電腦 GUI 大眾普及
- **1995 JavaScript** — 讓網頁互動成為可能

### Web 設計時代(1995-2007)
- **1990s 早期** — Jeffrey Zeldman、A List Apart、Web Standards Project 推動標準
- **Don Norman《設計心理學》**(1988)— 「user-centered design」奠基書;1993 Norman 在 Apple 提出「user experience」(UX)一詞
- **Jakob Nielsen《Usability Engineering》**(1995)— 把可用性工程系統化
- **2000-05 CSS + Flash 並存時代** — Flash 主導動態,CSS 主導內容
- **2004 Google AdSense, 2005 Web 2.0** — 從單向發佈到雙向互動

### 行動革命(2007-2014)
- **2007 [[iPhone]]** — 多點觸控 + 慣性滾動 + App 圖示矩陣,徹底改變 UI 範式
- **2008 App Store + Android Market** — APP 經濟誕生
- **2011-13 Apple iOS 7** — 從擬物化轉向扁平化(flat design)
- **2014 Google Material Design** — Google 全球設計系統,把「卡片」變介面通用語

### 介面成熟期(2014-2022)
- **設計系統(Design System)** — Atlassian、Airbnb、IBM Carbon、Salesforce Lightning 等
- **暗色模式(Dark Mode)普及** — 2018 後成 OS 標配
- **Figma 上線**(2016)— 多人即時協作介面設計,取代 Sketch
- **語音介面** — Alexa、Siri、Google Assistant
- **AR/VR** — Oculus、Apple Vision Pro(2024)

### AI 介面時代(2022-)
- **2022 ChatGPT + Stable Diffusion + Midjourney** — 對話式介面與生成式設計
- **AI-first 應用** — Notion AI、Cursor、Linear AI、Arc Browser
- **設計與 AI 協作** — Figma AI、Adobe Firefly、Galileo 等,設計師工作流變化

## 此領域關鍵作品
- [[Macintosh]](1984)— GUI 大眾化起點
- [[iPhone]](2007)— 多點觸控革命
- iOS 7(2013, 設計師 Jony Ive 主導)— 扁平化典範
- Material Design(2014)— 全球設計系統

## 對其他設計領域的影響
- **數位 → 物理回流** — iOS 介面美學影響後續工業設計,讓「螢幕審美」反向定義實體產品
- **設計系統化** — UI/UX 開創的「Design System」方法現在應用到品牌識別、空間設計
- **使用者中心思維** — Don Norman 的方法擴散到建築、服務、政策設計
