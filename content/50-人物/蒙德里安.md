# 蒙德里安

> 荷蘭新造型主義藝術家

## 簡述

彼得·科內利斯·蒙德里安，通稱皮特·蒙德里安，是一名荷蘭畫家，風格派運動幕後藝術家和非具象繪畫的創始者之一，對後代的建築、設計等影響很大。自稱“新造型主義”，又稱“幾何形體派”。


## 生平
1872 生於荷蘭 Amersfoort,父親是新教學校校長兼業餘畫家。早年繪自然寫實風景與荷蘭典型風車。1908 接觸神智學家舒恩馬克斯(Schoenmaekers)的《新造形主義》理論。1912 移居巴黎,接觸畢卡索與勃拉克立體派,把樹、海堤、教堂等對象逐步抽象化(著名的「樹系列」1908-13)。1914 一戰爆發回荷蘭,1917 與[[特奧·凡·杜斯伯格]]共同創辦《De Stijl》雜誌,進入純粹幾何抽象階段。1919 返巴黎,1921 完成第一張完整「紅黃藍構成」。1925 杜斯伯格在課程中引入 45 度斜線,蒙德里安認為違反新造形主義教條與其決裂,退出運動。1938 二戰前夕避難倫敦、1940 移居紐約。紐約期間受爵士樂衝擊,畫出晚期傑作《百老匯爵士樂》(Broadway Boogie Woogie, 1942-43)— 把規律網格打散成跳動的方塊節奏。1944 紐約因肺炎辭世。

## 設計理念
- **新造形主義(Neo-Plasticism)** — 藝術應拋棄所有「個體情感」與「自然模仿」,只保留宇宙的純粹結構
- **三原色 + 黑白灰** — 任何其他顏色都是「不純粹」,只用紅、黃、藍 + 黑、白、灰
- **絕對水平與垂直** — 斜線是運動 = 個體情感,被禁止;水平垂直是宇宙永恆秩序
- **不對稱平衡(Dynamic equilibrium)** — 不用對稱卻達到視覺平衡,動態而非靜止
- **「色彩 = 物質、線條 = 靈魂」** — 神智學影響的形而上學立場
- **晚期變化** — 紐約時期受爵士樂打散網格,把彩色色塊本身變成節奏單位

## 代表作品
- 《灰樹》(Gray Tree, 1911-12)— 寫實到抽象的過渡作
- 《構成 II》(Composition II in Red, Blue, and Yellow, 1930)— 紅黃藍時期經典
- 《百老匯爵士樂》(Broadway Boogie Woogie, 1942-43, MoMA 藏)— 晚期傑作
- 《勝利爵士樂》(Victory Boogie Woogie, 未完成, 1944)— 死前最後作品
- 1942 紐約畫室現場 — 牆上貼滿色卡的工作環境本身被拍照,成為設計師朝聖檔案

## 影響
- **[[風格派]]** — 蒙德里安的繪畫是運動視覺語言的核心,從建築([[里特費爾德施羅德住宅]])到家具([[紅藍椅]])皆其延伸
- **[[包浩斯]]** — 1922 杜斯伯格訪威瑪後,蒙德里安繪畫的「水平垂直 + 三原色 + 黑白」直接成為包浩斯預備課程的視覺語彙
- **[[國際樣式]]** — 密斯[[巴塞隆納德國館]]的「板片結構 + 流動空間」直接是蒙德里安繪畫的三維延伸
- **平面設計** — [[瑞士國際主義]]的網格系統繼承蒙德里安的「線分割平面」邏輯
- **時尚** — Yves Saint Laurent 1965「蒙德里安裙」把繪畫變成成衣;後續 Nike、Vans 反覆復刻
- **流行文化** — 紅黃藍 + 黑線的「蒙德里安式」變成識別性極高的視覺速記,印在 T-shirt、咖啡杯、iPhone 殼到無止盡
- **CSS Grid 美學** — 當代網頁設計的彩色區塊組合(尤其 brutalist web design)是蒙德里安的螢幕版
