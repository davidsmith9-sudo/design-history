## 📊 各類筆記數量（自動）

```dataview
TABLE length(rows) AS "筆數"
FROM "10-時代" OR "20-地區" OR "30-領域" OR "40-流派" OR "50-人物" OR "60-作品" OR "70-理論"
WHERE type
GROUP BY type
```

---

## 📊 所有流派（依年代）

```dataview
TABLE
  era AS "時代",
  region AS "地區",
  year_start AS "起",
  year_end AS "迄",
  status AS "完整度"
FROM "40-流派"
WHERE type = "流派"
SORT year_start ASC
```

---

## 📊 待補完的筆記（status = stub）

```dataview
LIST
FROM "10-時代" OR "20-地區" OR "30-領域" OR "40-流派" OR "50-人物" OR "60-作品" OR "70-理論"
WHERE status = "stub"
SORT file.name ASC
```

## 資料夾說明

| 資料夾 | 內容 |
|---|---|
| `00-總覽` | 入口、年表、地圖 |
| `10-時代` | 各時代總述 |
| `20-地區` | 各地區發展 |
| `30-領域` | 平面、工業、建築…… |
| `40-流派` | 包浩斯、風格派…… |
| `50-人物` | 設計師、理論家 |
| `60-作品` | 具體作品 |
| `70-理論` | 重要文獻、概念 |
| `99-素材` | 圖片、PDF、附件 |
| `_templates` | 筆記範本 |
