# 歐洲設計史

> **主要時代**　古代 → 當代（設計史核心舞台）

設計史的核心舞台：古希臘羅馬古典、中世紀哥德、文藝復興、工業革命、現代主義皆在此誕生。重要設計學校與運動多源於此。

## 此地區的流派

- [[工藝美術運動]]（英國，1860–）
- [[新藝術運動]]（法國/比利時/西班牙，1890–）
- [[維也納分離派]]（奧地利，1897–）
- [[構成主義]]（蘇聯，1913–）
- [[達達主義]]（瑞士/德國，1916–）
- [[風格派]]（荷蘭，1917–）
- [[包浩斯]]（德國，1919–1933）
- [[裝飾藝術]]（法國，1920–）
- [[國際樣式]]（德國 → 全球，1920s–）
- [[瑞士國際主義]]（瑞士，1950s–）
- [[解構主義]]（歐美建築，1980–）

## 設計特徵

- 古典秩序不斷回歸（古希臘羅馬 → 文藝復興 → 新古典）
- 設計學校制度發達（包浩斯、烏爾姆、皇家設計學院）
- 工藝 / 工業 / 藝術的辯論貫穿歷史
- 北歐功能主義 vs 義大利表現主義的雙極張力
- 政治 / 宗教 / 學院深度形塑風格走向

## 重要主題

- 古典與哥德建築（希臘 / 羅馬 / 法國）
- 工業革命與設計（英國）
- 包浩斯與德國設計
- 義大利設計（戰後家具、孟菲斯）
- 北歐設計（功能主義、實用主義）
- 瑞士平面設計（網格、Helvetica）
