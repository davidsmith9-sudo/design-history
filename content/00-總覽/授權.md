# 授權與引用

> 本站關於使用、引用、版權的完整聲明。

## 文字內容

本站所有**原創文字**(條目敘述、編輯註、方法論、本頁)以
**[Creative Commons Attribution-ShareAlike 4.0 International](https://creativecommons.org/licenses/by-sa/4.0/deed.zh_TW)** (CC BY-SA 4.0) 釋出。

意思是 ——

- **可以**:自由分享、改作、商業使用
- **必要**:
  - 註明原作者(本站)
  - 提供授權連結
  - 標示是否做過更動
  - **若改作後再分享,必須沿用相同授權 (ShareAlike)**

## 圖片

文章內圖片**版權歸原作或機構所有**。本站僅在「教育、評論、研究」範疇內使用,以中華民國《著作權法》第 52 條(合理使用 / fair use)為法律基礎。

- 多數圖片來自 **Wikipedia / Wikimedia Commons**(公有領域或 CC 授權)
- 部分圖片來自 **建築事務所 / 博物館官方發布**(版權保留)
- 部分圖片來自 **設計史教科書 / 學術論文**(評論引用)

每篇文章末尾的「**Sources & Credits**」區會列出該條目的主要來源連結,讀者可順藤摸瓜回溯原始版權頁。

> **若你是原作者或版權持有方**,看到本站使用了你的素材且希望調整(註明、移除、修改),請依下方「訂正通報」聯繫,我會在 48 小時內處理。

## 引用本站

寫論文 / 報告 / 文章引用本站時,**首選**透過詳情頁右下角的「引用 · Cite」按鈕,**會自動產生** Chicago / APA / MLA / BibTeX 四種格式。

### 範例:Chicago 17th

> "包浩斯." 設計史筆記 Design History Notes. Last modified 2024. https://design-history.pages.dev/40-流派/包浩斯.

### 範例:APA 7th

> 設計史筆記. (2024). 包浩斯. 設計史筆記 Design History Notes. https://design-history.pages.dev/40-流派/包浩斯

### 範例:BibTeX

```bibtex
@misc{Bauhaus2024,
  title  = {包浩斯},
  author = {設計史筆記},
  year   = {2024},
  url    = {https://design-history.pages.dev/40-流派/包浩斯},
  note   = {accessed 2024-XX-XX}
}
```

## 內容狀態(Status)

每篇條目右上角的徽章標示**完成度**,引用前請留意:

- **● Full · 完整** —— 內容已多輪校訂,可放心引用
- **◐ Stub · 雛形** —— 結構完整但篇幅短,可引用但建議交叉比對其他來源
- **○ Draft · 草稿** —— 進行中,**不建議**作為唯一來源

## 訂正通報 / 學術質疑

若你發現:

- 事實錯誤(年份、地點、人名、作者)
- 詮釋偏差(觀點 / 流派歸屬有爭議)
- 圖片版權問題
- 翻譯不準確
- 連結失效

歡迎透過以下任一方式回報:

- **GitHub Issue**(首選,可追蹤): [github.com/davidsmith9-sudo/design-history/issues](https://github.com/davidsmith9-sudo/design-history/issues)
- **Email**: *(填入聯絡 email)*
- **Pull Request**:直接改 markdown 提 PR,我會 review

修正後條目下方的「Last updated」會更新。
